Random variavbles

Discrete Random Variables

cumulative distribution function (CDF):
$F(x)=P(X ≤ x)$

  • mean
    $\mu =E(X)=\sum_{all x}^{}xP(X=x)$

    NOTE: 4 main statistical usages of the word ‘mean’:

    • The mean of a sample: $\overline x$
    • The mean of a population: $\mu$
    • The mean of a random variable :X
    • The distribution of the sampling mean: $\overline X$
  • expectation
    $E(g(x))=\sum_{all x}^{}g(x)P(X=x)$
  • variance
    $\sigma^2 =Var(x)=E(X-\mu)^2=E(X^2)-E(X)^2$

Binomial Distribution

$X~Bin(n,p)$
$P(X=x)=(_x^n)p^x(1-p^{n-x}$

  • mean: E(X)=np
  • variance: Var(x)=np(1-p)

    Hypergeometric distribution

    without replacement
    $P(X=x)=\frac{ (_x^{N_1})(_{n-x}^{N_2}) }{(_n^N)}$

Continuous Random Variables

Normal distribution

$X~N(mean,sd^2)$
$f(x)=\frac{1}{ \sqrt{(2\pi \sigma ^2) }}e^{-\frac{(x-\mu)^2}{2\sigma ^2}}$

$F(x)=P(X\leq x)=\int_{-\infty}^{x}f(y)dy$

Standard Normal

$Z~N(0,1)$

$P(X\leq x)=P(Z\leq \frac{x-\mu}{\sigma}) $

Student’s t-distribution

X ~ $t_n$, with n degrees of freedom

mean: 0
variance: $\frac{n}{n-2}$ for n>2

Chi-Squared distribution

X ∼ $x_n^2$ where n is the degrees of freedom

mean:n
variance: 2n

F-Distribution

X ∼ $F_{m,n}$ scaled ratio of two independent χ2 random variables, with m and n degrees of freedom.

Combinations of Random Variables

Linear Function of a Random Variable

$ Y = a + bX$

$ E(Y ) = a + bE(X)$

$Var(Y ) = b^2Var(X)$

$X∼N(\mu,\sigma^2) —> Y~N(a+b\mu,b^2\sigma^2)$

Independence for Random Variables)

$P(X\leq x,Y \leq y)=P(X \leq x)P(Y \leq y)$

  • correlation
    $Cov(X,Y)=E(XY)-E(X)E(Y)$

Central Limit Theorem (CLT)

If $X_i$ ∼ $(μ,σ^2)$ for i = 1,2,…,n then

$\overline X \approx N(\mu,\frac{\sigma^2}{n})$

  • normal dis (N>=30)
    $\overline X ~ N(\mu,\frac{\sigma}{\sqrt{n}})$

  • binomial dis (n>25,np>5,nq>5)
    approximate: $\overline p \approx N(p, \sqrt{ \frac{pq}{n} })$

    • bin -> normal
      if X~Bin(n,p) when n is large (n>25,np>5,nq>5)
      -> $Y~N(\mu=np,\sigma=\sqrt{npq})$
