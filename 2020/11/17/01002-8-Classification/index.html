Simple Linear Regression

$Y_i=\alpha+\betaX_i+\varepsilon_i)$
Error/residual:
$\varepsilon=SSE=\sum(y_i-\hat{y_i})^2$

Ordinary Least Squares (OLS):

find minimum $\frac{1}{n}\sum_{i=1}^n ((\hat{\alpha}+\hat{\beta}x_i)-y_i)^2 $

$\hat{\beta}=\frac{(x_i-\overline{x})(y_i-\overline{y})}{(x_i-\overline(x)^2}=\frac{cov(x,y)}{Var(x)}=\frac{r(x,y)*sd(y)}{sd(x)}$

$\hat{\alpha}=\overline{y}-\hat{\beta}\overline (x)$

Coefficient of determination (R2)

$R^2=1-\frac{\sum(y_i-\hat{y_i})^2}{\sum(y_i-\overline{y_i})^2}=1-\frac{SSE}{SST}$

Standard error (S)

$S=\sqrt(\frac{SSE}{N})$

Multiple Linear Regression

Gradient descent
