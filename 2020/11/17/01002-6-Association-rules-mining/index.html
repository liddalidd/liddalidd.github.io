Reducing the number of candidates

  • Apriori Principle
    If an itemset is frequent, then all of its subsets are also frequent

  • Conversely
    If an itemset is infrequent, then its supersets are also infrequent

Rule Generation

Given a frequent itemset L, find all non-empty subsets such that satisfies the minimum confidence requirement
If |L| = k, then there are candidate association rules.
