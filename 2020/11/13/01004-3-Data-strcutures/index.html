Stack

last-in-first-out (LIFO)
Each operation runs in time O(1)

  • push(e): inserts an element, e
  • pop(): removes and returns the last inserted element

Queue

first-in-first-out (FIFO)

  • enqueue(e): inserts an element, e, at the end of the queue
  • dequeue(): removes and returns element at the front of the queue

Priority Queue ADT

Selection-Sort

  1. inserting elements with n insert operations takes O(n) time
  2. removing elements with n remove_min operations takes O(n2)

Insertion-Sort

  1. inserting elements with n insert operations takes O(n2) time
  2. removing elements with n remove_min operations takes O(n)

Heap (min-heap)

Heap-Order: for every node m ‚â† root, $key(m)\geq key(parent(m))$
A heap storing n keys has height log n

  • Insertion O(log n)
  • Removal O(log n)

Heap-Sort

the version of priority-queue sorting that implements the priority queue with a heap. It runs in O(n log n) time.

Trees

Tree Terminology

  • Depth of a node:number of ancestors not including itself
  • Level: set of nodes with given depth
  • Height of a tree: maximum depth

Traversing trees

  • pre-order
    father-son1-son2
    on the left
  • post-order
    son1-son2-father
    on the right
  • in-order (for binary trees)
    left son-father-right son
    from below

Binary Search Tree (BST)

an inorder traversal of a binary search tree visits the keys in increasing order.
key(left descendant) < key(node) < key(right descendant)

  • search
    Runs in O(h) time, where h is the height of the tree
    worst case is h=n (like a list)
    best case is h=log2 n (like a tree)
  • insertion
  • delete
    • w has one external child
    • w has two internal children
      replace with the smallest key among the right subtree

Improving Balance: Trinode Restructuring

  • Single Rotations
    O(1) time
  • Double rotations
    O(1) time

Rank-balanced Trees

AVL Trees

The ranks of the two children of every internal node differ by at most 1.
The height of an AVL tree storing n keys is O(log n).

  • Insertion
    • insert to have BST property, re-arrange to AVL
  • Removal
    • remove to have BST property, re-arrange to AVL
Method Time
space O(n)

Insertion O(log n)
