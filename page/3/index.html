Discrete Random Variables

cumulative distribution function (CDF):
$F(x)=P(X ≤ x)$

  • mean
    $\mu =E(X)=\sum_{all x}^{}xP(X=x)$

    NOTE: 4 main statistical usages of the word ‘mean’:

    • The mean of a sample: $\overline x$
    • The mean of a population: $\mu$
    • The mean of a random variable :X
    • The distribution of the sampling mean: $\overline X$
  • expectation
    $E(g(x))=\sum_{all x}^{}g(x)P(X=x)$
  • variance
    $\sigma^2 =Var(x)=E(X-\mu)^2=E(X^2)-E(X)^2$

Binomial Distribution

$X~Bin(n,p)$
$P(X=x)=(_x^n)p^x(1-p^{n-x}$

  • mean: E(X)=np
  • variance: Var(x)=np(1-p)

    Hypergeometric distribution

    without replacement
    $P(X=x)=\frac{ (_x^{N_1})(_{n-x}^{N_2}) }{(_n^N)}$

Continuous Random Variables

$X~N(mean,sd^2)$
$f(x)=\frac{1}{ \sqrt{(2\pi \sigma ^2) }}e^{-\frac{(x-\mu)^2}{2\sigma ^2}}$

    $P(A`)=1-P(A)$

  • mutually exclusive
    $P(A\cap B)=0$
    $P(A\cup B)=P(A)+P(B)$

  • independence
    $P(A\cap B)=P(A)P(B)$
    $P(A|B)=P(A)$

  • union rule
    $P(A\cup B)=P(A)+P(B)-P(A\cap B)$

  • conditional probility
    $P(A|B)=\frac{P(A\cap B)}{P(B)}$

  • Multiplication rule
    $P(A\cap B) = P(A|B)P(B)$
    Joint probability = conditional probability     P(condition)

  • Bayes theoren
    $P(A|B)=\frac{P(B|A)P(A)}{P(B)}$

  • the law of total probabilitiy

    OPT(j) = value of optimal solution to the problem consisting of job requests 1, 2, …, j.

  2. Find recurrences

  • Case 1: OPT selects job j.
  • Case 2: OPT does not select job j.
    OPT(j) = max {vj + OPT (p(j)), OPT(j-1)}
  1. Solve the base cases
    OPT(0) = 0

Divide and conquer

Divide and Conquer algorithms can normally be broken into
these three parts:

  1. Divide If it is a base case, solve directly, otherwise break up the problem into several parts.
  2. Recur Recursively solve each part [each sub-problem].
  3. Conquer Combine the solutions of each part into the overall
    solution.

Binary search

T(n)=O(logn)

Sorting

Merge-Sort

T(n)=O(nlogn)

Solving recurrences by unrolling

General strategy:

  • Analyze first few levels
  • Identify the pattern for a generic level
  • Sum up over all levels
